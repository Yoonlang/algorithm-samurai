### 두 번째 풀이

한 줄로 줄여보고 싶어서 새로 작성했는데, 첫 번째 풀이에 비해 메모리를 비교적 많이 잡아 먹는다.

`trim()` 호출 시 새로운 문자열을 생성하기 때문이다.

```js
/**
 * @param {string} s
 * @return {number}
 */
let lengthOfLastWord = function (s) {
  return s.trim().split(" ").pop().length;
};
```

### 첫 번째 풀이

```js
/**
 * @param {string} s
 * @return {number}
 */
let lengthOfLastWord = function (s) {
  s = s.split(" ").filter((v) => v !== "");
  return s[s.length - 1].length;
};
```
